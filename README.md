# Football Ticket Booking System

Simple SQL assignment: 3 tables (`Users`, `Matches`, `Bookings`) + 7 queries.

## Folder Structure

- `schemas/create_tables.sql` — creates the 3 tables (PK/FK + constraints)
- `schemas/seed_data.sql` — inserts sample data
- `queries/query_01.sql` to `query_07.sql` — each query, one per file
- `QUERY.sql` — all 7 queries together

## How to Run

1. Run `schemas/create_tables.sql`
2. Run `schemas/seed_data.sql`
3. Run `QUERY.sql` (or individual files in `queries/`)
